About the review of the application

This is called the Acceptance stage. This when the developer sends us all of the information in their application, including all of the consultation they have done. We look at all of the documents to check if we can accept the application for Examination.

This stage can have one of three outcomes:

  • we accept the application for examination
  • we do not accept the application
  • the developer withdraws their application

We have 28 days to make the decision from when the developer sends us the application. If it is accepted, this does not mean the development can go ahead, this means we’ve accepted the application for examination.

What the Planning Inspectorate does to review the application

Our team will check the application to make sure the Examining Authority has everything it needs to be able to properly examine the project at the Examination stage.

If there is anything missing or we need more information, we may not be able to accept it for Examination.
