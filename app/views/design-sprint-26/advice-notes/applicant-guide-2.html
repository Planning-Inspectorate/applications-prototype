Submitting an application for development consent

Applicants are advised to contact the Planning Inspectorate before completing the form to ensure that electronic documentation is provided in the required manner.

The Planning Inspectorate’s Advice Note Six: Preparation and submission of application documents provides more information in this regard.

The application form contains functionality that is not supported by all browsers and is best completed using adobe reader. Clicking on the form link allows you to save the file or open it in adobe reader. Should you have any difficulties downloading the form please email Corpcomms@planninginspectorate.gov.uk

Adobe reader can be downloaded at http://www.adobe.com/products/reader/

Please note that guidance is also embedded in the application form itself.
